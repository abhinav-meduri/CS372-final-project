# Videos

This directory contains the demo and technical walkthrough videos for the Patent Novelty Assessment System.

## Required Videos

### 1. Demo Video
- **Purpose:** Demonstrate the application's functionality and user interface
- **Content:** Show the Streamlit app in action, entering queries, viewing results, and interpreting outputs
- **Duration:** Recommended 3-5 minutes
- **Filename:** `demo.mp4` (or upload to YouTube/Vimeo and link in main README)

### 2. Technical Walkthrough
- **Purpose:** Explain the technical approach, methodology, and implementation
- **Content:**
  - System architecture (hybrid retrieval, ML classification, LLM explanations)
  - Feature engineering approach (10 features)
  - Model training and evaluation
  - Key design decisions and trade-offs
- **Duration:** Recommended 5-10 minutes  
- **Filename:** `technical_walkthrough.mp4` (or upload to YouTube/Vimeo and link in main README)

## How to Add Videos

**Option 1: Store locally in this directory**
```bash
# Place video files here
videos/demo.mp4
videos/technical_walkthrough.mp4
```

**Option 2: Host externally and link in README**
1. Upload videos to YouTube, Vimeo, or Duke Box
2. Update the "Video Links" section in `README.md` with the URLs

## Video Guidelines

- **Quality:** 1080p minimum resolution recommended
- **Audio:** Clear narration with minimal background noise
- **Format:** MP4 (H.264 codec) for compatibility
- **Size:** If hosting in repo, keep under 100MB per video (use compression if needed)
